## 📌 Descripción General

El programa realiza y visualiza operaciones fundamentales de la **Teoría de la Computación** relacionadas con cadenas y lenguajes formales.

---

## 🖥️ Funcionalidades

### 1. Subcadenas, Prefijos y Sufijos

✔ Generación de todas las **subcadenas** posibles de una cadena.  
✔ Obtención de **prefijos**.  
✔ Obtención de **sufijos**.  
✔ Visualización de los resultados.  
✔ Exportación de resultados a archivo de texto.

---

### 2. Cerradura de Kleene (Σ*) y Cerradura Positiva (Σ+)

✔ Generación de Σ* (incluye la cadena vacía ε).  
✔ Generación de Σ+ (excluye la cadena vacía).  
✔ Definición del alfabeto por el usuario.  
✔ Selección de longitud máxima de generación.  
✔ Exportación de resultados.

---

## ⚙️ Requisitos del Sistema
Debes de contar con lo siguiente para poder hacer uso del programa: 
- ☕ Java JDK 8 o superior  
- 💻 Visual Studio Code  
- 📦 Extensión **Extension Pack for Java** instalada  
- 🖥️ Windows, Linux o macOS

## 🚀 Instrucciones de Ejecución

### 🔹 Opción 1: Ejecutar desde IDE NetBeans 

1. Clonar o descargar el repositorio.
2. Abrir NetBeans
3. Seleccionar File → Open Project
4. Buscar la carpeta del proyecto.
5. Seleccionar el proyecto y hacer clic en Open Project.
6. Ejecutar el programa presionando: Botón Run, o Tecla F6.

### 🔹 Opción 2: Ejecutar desde Visual Studio Code

1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en **Visual Studio Code**.
3. Verificar que la extensión **Extension Pack for Java** esté instalada.
4. Abrir el archivo **interface.java** que contiene el método:
   ```java
   public static void main(String[] args)
5. Hacer clic en el botón Run (▶) que aparece sobre el método main
